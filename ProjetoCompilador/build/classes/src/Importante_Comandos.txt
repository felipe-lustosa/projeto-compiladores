
Sequência de comandos para o Terminal de Comandos:

javac -encoding UTF8 COMP.java  // É responsável pela compilação
java COMP.java			// É responsável pela execução



Para os testes:

java COMP -a "E:\Documentos\UNIVASF_2022_2\Compiladores\ProjetoCompilador\Projeto\Exemplos_de_teste\Exemplo0_ok.txt"

Obs.: O endereço deve ser colocado no Terminal de Comandos sem as aspas!
Obs.: É importante alterar a localização do exemplo de teste, de acordo com a localização em sua máquina!
